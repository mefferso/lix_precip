WFO LIX Station Observation Feeds

This page is the boring-but-useful backbone for a future map UI. Right now it publishes fresh CSV/JSON products for 24-hour rainfall, latest temperature, daily minimum temperature, and daily maximum temperature.

Data source: Synoptic Weather API. Files below are rebuilt by GitHub Actions.
